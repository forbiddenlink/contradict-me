## 💬 What It Does

State your opinion on any topic — ContraDict Me uses AI to challenge your position with counter-arguments, evidence, and logical rebuttals. Sharpen your critical thinking and debate skills.

## ✨ Features

- 🧠 AI-powered counter-argument generation
- 📝 Enter any position or opinion
- ⚖️ Receive steelman counter-arguments
- 💡 Learn to defend positions more rigorously

## 🚀 Getting Started

```bash
git clone https://github.com/forbiddenlink/contradict-me
cd contradict-me
npm install
cp .env.example .env.local
# Add your OPENAI_API_KEY to .env.local
npm run dev
```

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **AI:** OpenAI API
- **Styling:** Tailwind CSS
